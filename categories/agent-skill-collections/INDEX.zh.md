## 什么该放这里

一个刻意拥挤的领域——agent **技能 / 提示词 / subagent 人设 / harness 配置**的合集，没有哪一个是唯一答案。
按用途领域组织(长尾再按来源：厂商 vs 个人)，让 agent 能按任务和出处挑捆绑包。自平衡：某叶子超出 fanout 就继续拆。
