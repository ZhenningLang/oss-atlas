# frontend-animation

> 分类节点。面向 Web 的 JavaScript 动画引擎与运动库。

## 本分类项目

| 项目 | 何时用 | 页面 |
|---|---|---|
| **Anime.js** | 零依赖的 JS 动画引擎:统一 animate() API 驱动 CSS、SVG、DOM 属性和 JS 对象,内置时间线、错峰、弹簧缓动和滚动联动。 | [→](anime.zh.md) |

## 对比矩阵

| 选项 | 是否收录 | 一句话取舍 |

## 什么该放这里

**给 Web UI 做动画**的库——JS 运动引擎、时间线/补间系统。不含 agent 驱动的设计生成(见 `ai-design-generation`)。
